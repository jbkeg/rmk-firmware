# 한국어 레이아웃 가이드

이 문서는 RMK 기반 키보드에서 한국어 입력 운용 원칙을 정의합니다.

## 기본 원칙

- 펌웨어는 표준 HID 키코드만 전송합니다.
- 한글 조합/자모 처리는 호스트 OS IME가 담당합니다.
- 펌웨어에서 언어 전용 조합 로직을 유지하지 않습니다.

이 저장소는 OS 위임 입력 모델을 기본으로 합니다.

## 두벌식과 세벌식

### 두벌식 (기본)

![korean 2set](/docs/images/korean_2set_layout.svg)

- 모든 주요 OS에서 기본 지원
- 별도 설치 없이 사용 가능
- 기본 권장 조합: `QWERTY + 두벌식`

### 세벌식 (선택)

![korean 3set](/docs/images/korean_3set_layout.png)

- 고급 입력 방식
- 장시간 한국어 타이핑 사용자에게 유리할 수 있음
- OS 설정에서만 활성화

## 권장 조합

| 영문 배열 | 한글 배열 | 권장도 |
| --- | --- | --- |
| QWERTY | 두벌식 | 기본 권장 |
| Colemak | 세벌식 | 고급 사용자 |
| QWERTY | 세벌식 | 가능하나 비권장 |
| Colemak | 두벌식 | 비권장 |

## 한영 전환

한영 전환은 OS 단축키 호출 방식만 사용합니다.

- Windows: Right Alt(한/영)
- macOS: Caps Lock 또는 Control + Space

## 비목표

다음은 저장소 목표가 아닙니다.

- 펌웨어 기반 한글 조합
- 자모 단위 매크로
- IME 상태 추적/동기화

한국어 입력은 OS 레벨에서 관리하고, 펌웨어는 이를 방해하지 않는 방향을 유지합니다.
